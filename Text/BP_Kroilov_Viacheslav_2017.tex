\abstractEN{A numerical database speeds up computations by memoizing pairs of an argument and the result, computed by a function with the argument. A canonical numerical database is based on the weighted search tree~-- a combination of the AVL tree and the binary heap. An application of alternative data structures, namely hash table and splay tree, is discussed in this thesis. In addition, a new data structure~-- \cndcname{}~-- is introduced. It is similar to the weighted search tree, but all operations are declared as thread-safe.

            All mentioned above data structures are implemented in the C++ programming language as a programming library, called \numdbname. The performance of each data structure is measured and the results are compared and discussed.
}
\abstractCS{Numerická databáze zrychluje výpočet ukládáním mezivýsledků do pamětí.
            Kanonická implementace numerické databáze je založená na ohodnoceném
            binárním stromu~-- kombinace AVL-stromu a binární haldy.
            V této práci je diskutována i možnost využití jiných datových struktur,
            jak je Splay-strom a hašovácí tabulka.
            Navíc je zavedená zcela nová datová struktura~-- \cndcname. Podporuje stejné
            operace jako ohodnocený binární strom,
            ale je přizpůsobená k použití ve vícevláknovém prostředí.

            Všechny zmíněné datové struktury jsou implementovány v
            programovacím jazyce C++ v podobě programovací knihovny \numdbname.
            Na závěr jsou uvedené výsledky měření výkonnosti implementovaných
            datových struktur.
}
